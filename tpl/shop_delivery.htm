Отправка производится транспортными компаниями: "Новая почта", "Автолюкс", "Ночной Экспресс", а также "Укрпочтой" в населенные пункты, где нет представительств других компаний.

Возможна доставка наложенным платежом через представительства "Новой почты" и "Автолюкса".

Варианты оплаты:

1. По перечислению (без НДС для физических лиц и юридических лиц не плательщиков НДС).

2. По перечислению ( с НДС для юридических лиц плательщиков НДС).

3. По перечислению на карточный счет через терминалы "Приватбанка" ( в том числе через платежную систему Ligpay).

4. Наложенным платежом при получении.
